# 국내 ETF 자산배분 백테스트 (Backtest v3.0)

국내 상장된 ETF들을 활용하여 다양한 자산배분 전략을 백테스트하고 성과를 분석하는 도구입니다.

## 주요 기능
- **데이터 로드**: `etf_data.xlsx` 파일을 통해 국내 ETF 정보를 관리하고 분석에 활용합니다.
- **백테스트 엔진 (`backtest.py`)**:
    - `yfinance`를 이용한 실시간(또는 과거) 가격 데이터 다운로드.
    - 월간(Monthly), 분기(Quarterly), 연간(Yearly) 리밸런싱 및 Buy & Hold 전략 지원.
    - 성과 지표 산출: 누적 수익률, CAGR, 변동성, 샤프 지수, MDD 등.
    - 벤치마크(KODEX 200)와의 성과 비교 분석.
- **GUI 인터페이스 (`backtest_gui_1.py`)**:
    - 전략 프리셋 제공: 60/40 전략, 올웨더 포트폴리오(간소화), 영구 포트폴리오.
    - 사용자 정의 포트폴리오 구성 (Port A vs Port B 비교).
    - 시각화: 누적 자산 가치 그래프, 12개월 롤링 수익률, 낙폭(Drawdown) 분석.
    - 월별 수익률 매트릭스 제공.
    - 설정 저장 및 불러오기 (JSON 포맷).

## 설치 및 실행 방법

### 요구 사항
- Python 3.8 이상
- 필요 라이브러리:
  ```bash
  pip install pandas numpy matplotlib yfinance openpyxl
  ```

### 실행 방법
```bash
python backtest_gui_1.py
```

## 사용법
1. **종목 선택**: 하위 시장 및 그룹 필터를 통해 원하는 ETF를 선택하여 Port A 또는 Port B에 추가합니다.
2. **비중 설정**: 각 종목의 비중을 입력합니다. (자동 정규화 및 동일 비중 설정 기능 제공)
3. **백테스트 설정**: 시작일, 초기 금액, 무위험 수익률, 리밸런싱 주기 등을 설정합니다.
4. **실행**: '백테스트 실행' 버튼을 클릭하여 결과 리포트와 그래프를 확인합니다.

## 프로젝트 구조
- `backtest.py`: 백테스트 로직 및 데이터 처리 핵심 엔진.
- `backtest_gui_1.py`: `tkinter` 기반 사용자 인터페이스.
- `etf_data.xlsx`: 분석 대상 ETF 종목 마스터 데이터.
- `.gitignore`: 불필요한 파일(캐시, 가상 환경 등) 제외 설정.
